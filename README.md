# Personal Finance Tracker

A Python command-line application for managing personal finances using SQLite.

## Features

- Add Transactions
- View Transactions
- Update Transactions
- Delete Transactions
- Calculate Balance
- SQLite Database Storage

## Technologies

- Python
- SQLite
- Git

## Run

```bash
python main.py
```
